À propos

Cette API a été créée par la société Serli. Elle permet d'accéder aux données Open Data de Grand Poitiers. La première version de cette API est destinées à l'application iOS Poitiers Vélo.

La seconde version de cette API contient plus de source :

  • Les Stationnement vélos
  • Les Bornes à verre
  • Les Places handicapées GIG / GIC
  • Les Défibrillateurs

Des sources de données seront ajoutées au fil du temps.

Vous pouvez contribuer à cette API sur GitHub : https://github.com/Serli/opendata-poitiers-backend
